# 📌 Project Overview

This project is a **professional multi-page Power BI dashboard** developed to analyze Amazon E-Commerce sales performance.

The dashboard transforms raw sales data into meaningful business insights through interactive visualizations, KPIs, slicers, maps, and advanced DAX measures.

It demonstrates end-to-end Business Intelligence skills including:

- Data Cleaning
- Power Query
- Data Modeling
- DAX
- Dashboard Design
- Business Insights

# 💡 Key Insights

- Electronics generated the highest revenue.
- West region contributed the maximum sales.
- Express Shipping delivered faster than other shipping modes.
- UPI was the most preferred payment method.
- Return Rate remained below target.
- High discounts reduced profit margins.
