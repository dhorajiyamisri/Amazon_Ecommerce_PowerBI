# 📌 Project Overview

This project presents a **professional multi-page Power BI dashboard** built to analyze Amazon E-Commerce sales performance. It transforms raw transactional data into interactive business insights, helping decision-makers monitor sales, profitability, customer behavior, product performance, shipping efficiency, and executive-level KPIs.

The dashboard combines **Power Query**, **Data Modeling**, **DAX Measures**, and **interactive visualizations** to create an end-to-end Business Intelligence solution suitable for real-world reporting.

# 🛠 Technical Implementation

This project follows a complete Business Intelligence workflow, starting from raw data preparation to interactive dashboard development and business insight generation.

---

# 🔄 Data Preparation

The raw Amazon E-Commerce dataset was transformed using **Power Query Editor** before loading it into the data model.

### Data Cleaning Tasks

- ✅ Removed duplicate records
- ✅ Handled missing values
- ✅ Renamed columns
- ✅ Corrected data types
- ✅ Removed unnecessary fields
- ✅ Created calculated columns
- ✅ Standardized categorical values

---

# 🧩 Data Modeling

A structured data model was created to improve report performance and enable accurate calculations.

### Model Features

- Star Schema Design
- Fact & Dimension Tables
- One-to-Many Relationships
- Optimized Filtering
- Improved Query Performance
